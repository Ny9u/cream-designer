# Cream-Designer 表单生成器
#### 一款所见即所得的表单设计器，支持拖拽式生成表单，个性化自定义组件属性✨

### 功能一览
```
> 拖拽式表单设计模式；
> 支持实时加载表单；
> 支持表单复杂交互控制；
> 支持自定义组件样式；
> 支持自定义校验逻辑；
> 可导出Vue组件、Json代码；
> 未完待续...
```

### 安装依赖
```
npm install 
```

### 开发调试
```
npm run dev
```

### 生产打包
```
npm run build
```

### 服务端渲染
```
npm run dev:ssr
```
